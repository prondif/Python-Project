# Warehouse Management System (Tier 3)

This program simulates an automated warehouse system using
individual item tracking together with ADS communication.

## Features
- Individual item tracking
- Unique item IDs
- Automatic pallet movement
- Imaging and transfer operations
- Storage handling
- Automatic return to home position
- Object-oriented warehouse structure

## How it works
Each item is stored as an individual object with its own ID.

Example item IDs:
- BOX001
- BOX002
- BOX003

The simulator communicates with Python through ADS signals
to automate warehouse operations.

## Workflow
HOME -> IMAGING -> TRANSFER -> STORAGE -> HOME

## How to run
Run the file:

tier3_wms.py

Then follow the terminal instructions.
